# NJU 论文模板编译说明（Linux）

## 环境信息

- 系统：Ubuntu 20.04 LTS
- TeX Live 2026，安装路径：`~/texlive/2024/`
- 已配置 `~/.zshrc`，新开终端即可直接使用

## 日常编译

在模板目录下运行：

```bash
cd ~/Desktop/Thesis/NJU/doc/linux/latex

# 完整编译（含参考文献，推荐）
latexmk -xelatex njuthesis-sample.tex

# 清理辅助文件
latexmk -c
```

## 编译原理

完整编译需执行四步，`latexmk` 会自动完成：

```
XeLaTeX → Biber → XeLaTeX → XeLaTeX
```

手动执行：

```bash
xelatex njuthesis-sample.tex
biber njuthesis-sample
xelatex njuthesis-sample.tex
xelatex njuthesis-sample.tex
```

## 关键注意事项

1. **必须用 XeLaTeX**，不能用 pdfLaTeX（中文支持需要）
2. **字体设置**：Linux 下使用 `fontset = fandol`（Windows 为 `win`，macOS 为 `mac`）
3. **文件编码**：必须是 UTF-8

## 环境安装记录

Ubuntu 20.04 自带 TeX Live 2019，与 njuthesis v1.4.3（2025）不兼容，需从 TUG 官网安装新版：

```bash
# 下载安装脚本（使用清华镜像）
cd /tmp
wget https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/tlnet/install-tl-unx.tar.gz
tar -xzf install-tl-unx.tar.gz
cd install-tl-*

# 创建安装配置（跳过文档和源码，节省约 3GB 空间）
cat > /tmp/texlive.profile << EOF
selected_scheme scheme-full
TEXDIR $HOME/texlive/2024
TEXMFCONFIG $HOME/.texlive2024/texmf-config
TEXMFHOME $HOME/texmf
TEXMFLOCAL $HOME/texlive/texmf-local
TEXMFSYSCONFIG $HOME/texlive/2024/texmf-config
TEXMFSYSVAR $HOME/texlive/2024/texmf-var
TEXMFVAR $HOME/.texlive2024/texmf-var
option_doc 0
option_src 0
EOF

# 安装（约 30-60 分钟）
./install-tl --profile=/tmp/texlive.profile \
  --repository https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/tlnet
```

安装完成后写入 `~/.zshrc`：

```bash
echo 'export PATH="$HOME/texlive/2024/bin/x86_64-linux:$PATH"' >> ~/.zshrc
source ~/.zshrc
```

## 可修改的文件

| 文件 | 用途 |
|------|------|
| `njuthesis-sample.tex` | 论文主体内容 |
| `njuthesis-setup.def` | 个人信息（姓名、学号、导师等） |
| `njuthesis-sample.bib` | 参考文献数据库 |

## 不可修改的文件

| 文件 | 用途 |
|------|------|
| `njuthesis.cls` | 模板核心文件 |
| `njuthesis-*.def` | 各论文类型定义 |
| `nju-*.pdf` | 校徽和校名图片 |
